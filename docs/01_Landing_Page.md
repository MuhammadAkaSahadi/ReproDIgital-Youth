ReproDigital Youth — Halaman Landing / Beranda
Dokumen ini mendeskripsikan UI/UX untuk halaman utama (Landing Page) platform ReproDigital Youth secara lengkap per section dan per breakpoint.

1. Hero Section
Desktop Layout
•	Full viewport height (min 600px, max 800px)
•	Two-column grid: 60/40 split
•	Background: Gradient Teal 50 ke White (atas ke bawah)
•	Elemen dekoratif: Organic shapes SVG di Coral 50 dan Teal 100

Kolom Kiri:
•	Heading (Display size): 'Wujudkan Masa Depanmu'
•	Subheading (Body Large): Deskripsi platform
•	Spacing: 24px antara heading dan subheading
•	CTA Primary: 'Mulai Belajar' — Coral 500, white text, h:48px, px:32px, rounded-md, shadow-md
•	CTA Secondary: 'Lihat Video' — White bg, Teal 600 text, border Teal 600, play icon
•	Trust indicators (margin-top 96px): 'Aman & Rahasia', 'Konselor Terlatih', 'Gratis Selamanya' — horizontal, 32px spacing

Kolom Kanan:
•	Hero illustration: Remaja Indonesia diverse, flat illustration, warm palette
•	Aspect ratio: 4:3
•	Floating card overlay (bottom-left): White bg, shadow-lg, '1.200+ Remaja Bergabung', avatars

Tablet (768px–1023px)
•	Hero height: 600px, split 55/45
•	Font sizes turun 1 step, CTA buttons: 44px height

Mobile (<768px)
•	Single column, hero height: auto (min 500px)
•	Urutan konten: Ilustrasi dulu (300px), kemudian teks
•	CTAs stack vertikal full width 48px
•	Trust indicators: Grid 2x2
